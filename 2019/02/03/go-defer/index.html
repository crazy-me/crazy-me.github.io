defer执行时机

  • return 语句执行完之后，如果有 defer 语句，再执行 defer 语句
  • 发生 Panic ，也会触发 defer 执行

defer执行顺序

多个defer出现时，会以”先进后出，后进先出”的规则来执行，类似于数据结构中栈的执行顺序.
